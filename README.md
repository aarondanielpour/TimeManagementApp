# TimeManagementApp
A simple ASP.NET core MVC web application to manage tasks and priorities.

# Main Features
- Add, edit, delete tasks using CRUD methods
- Mark tasks as completed
- Filter by priority or due date
- Dashboard summary of total, completed, and pending tasks
- Responsive Bootstrap UI


## Tech Stack
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server LocalDB
- Bootstrap 5
